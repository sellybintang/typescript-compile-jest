tutorial menggunakan typescript pada jest.Js

1. npm init
2. npm i --save-dev @jest/preset-env
3. npm install @babel/preset-env --save-dev (sebagai compiler antara typescript ke js)
4. kemudian ganti script pada package.json menjadi babel jest bisa dicek pada https://babeljs.io/setup#installation
5. buat file bable.config.json 
6. npm i --save-dev typescript (untuk depedensi penggunaan typescript)
7. npx tsc --init (untuk membuat projek otomatis baru tsc)
8. pada tsconfig.json dirubah script dari commonjs menjadi ES6 agar merubah jenis script menjadi lebih expresifs

2. TSC (typescript compiler ke bentuk js)
caranya:
- edit outdir yang ada di tsconfig.json
- buat file untuk hasil compiler sesuai nama di outdir
- buat command npx tsc
Apabila kita ingin mengecualikan file untuk dicompile kita bisa menggunakan 
(exclude atau include)
- exclude = untuk mengecualikan file yang dicompile (exclude : src/**/*)
- inclue = untuk mengikutkan file yang dicompile(include: tests/**/*)

3. Tipe Data Primitif (hampir sama dengan js)
pada saat membuat variabel harus dideklarasikan tipe datanya
contoh :
const name:string = 'Selly Bintang'
const age:number = 21
// tidak bisa number karena akan terbaca menjadi bilangan oktal atau bilangan berbasis 8, maka dari itu diganti string
const telp:string="087575843"
const isVip:boolean = true

4. ARRAY
  -  pada penggunaan ARRAY pun sama harus dideklarasikan pada saat menentukan tipe data of array, contoh :
    const nama_penduduk :string[]= ["Selly","Fajar", "Ajeng"]
    const umur_penduduk :number[]= [20,20,20]
    const hapus_penduduk :boolean[]=[false,true, false]
    console.info(nama_penduduk[0])
    console.info(umur_penduduk[2])
    console.info(hapus_penduduk[1])
- pada array yang tidak bisa dimodifikasi bisa menggunakan ReadOnlyArray
    contoh:
    const data_nama :ReadonlyArray<string>= ["Selly", "Bintang"]
    const data_umur :ReadonlyArray<number>= [20,21]
    // jika penulisan console menjadi satu maka hasil akan tergabung
    console.info(data_nama[0],data_umur[1])
- pada tuple yang tidak bisa dimodifikasi bisa menggunakan readonly dimana
    isi tipe bisa ditentukan lebih dari satu, contoh:
    const data_nama :readonly[string,number,boolean]=["Selly", 20, true]
    console.info(data_nama[1],data_nama[2],data_nama[0])

5. Tipe Data ANY
tipe data yang flexible dimana atribut yang menggunakan beberapa jenis tipe data tanpa deklarasi atau pengecekan. Hal tersebut direkomendasikan saat urgent
contoh:
const data_person:any={
            nama:"selly",
            umur:22,
            alamat:"Banyuwangi"
        }

6. Union Type
- tipe data yang bisa diatur secara bebas sesuai tipe data yang telah diatur 
contoh:
 let data_person : string|number|Boolean=
        "Selly"
        console.info(data_person)
        data_person=true
        console.info(data_person)
        data_person=44
        console.info(data_person)

- unio type yang menggunakan condition agar tidak terjadi miskom saat memasang default pada tipe data 
contoh :
 function person(value: number|string|boolean){
            if (typeof value === 'string'){
                return value.toUpperCase()
            
            }else if(typeof value === 'number'){
                return value +3
            }else{
                return !value
            }
        }
